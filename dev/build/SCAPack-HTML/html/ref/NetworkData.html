NetworkData

NetworkData
represents the calculated network parameters for a netlist.
NetworkData[rawdata, type]
represents a user specified network parameter dataset.
MORE INFORMATION
  • Use Normal to view the underlying data.
  • NetworkData[rawdata, type] accepts a matrix of size suitable to the type specified. The currently supported types and their square matrix sizes are listed below.
typeDimensionRepresents
"Z"arbitraryZ-parameters
"Y"arbitraryY-parameters
"G"2G-parameters
"H"2H-parameters
"ABCD"2ABCD parameters
