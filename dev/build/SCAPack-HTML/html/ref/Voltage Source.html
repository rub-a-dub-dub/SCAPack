Voltage Source

{"volt", {pos, neg}, "name", {}}
is a voltage source between the nodes pos and neg.
MORE INFORMATION
  • The current through the voltage source is an additional solver variable.
  • The terminal voltage chosen as a solver variable is arbitrarily pos, unless it is the datum node, in which case it is neg.
