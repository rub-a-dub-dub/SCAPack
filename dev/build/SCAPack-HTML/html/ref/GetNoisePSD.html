GetNoisePSD

GetNoisePSD[netlist, nodes, srcname]
returns the noise power spectral density between the specified nodes for the noise source srcname in the netlist.
MORE INFORMATION
  • GetNoisePSD[netlist, nodes, srcname] determines the output noise due to the noise source specified.
  • nodes is a list of two nodes, between which the output noise spectrum is calculated.
  • All independent sources are disabled prior to determining the output noise spectrum.
