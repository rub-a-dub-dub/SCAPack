Manually Solving Circuits

If so desired, users may manually solve a circuit by using SCAPack to convert a netlist into a system of equations. Routines to automatically determine the various variables or even to automatically execute Solve are available to aide the user.
NetlistCircuit[netlist]converts the netlist into a system of equations
GetSolveVariables[netlist]returns a list of all variables in the netlist
SolveFullSystem[netlist]automatically solves the netlist for all of the identified variables

Automated Solvers

SCAPack provides several methods to automatically determine performance metrics for a given netlist. Automation is provided for matters as simple as determining the transfer function of a netlist, to more advanced methods such as zero-value time constant analysis.
GetTransferFunction[netlist,{output,input}]returns the transfer function betweent the output and input variables
GetInputImpedance[netlist,{output,input}]computes the impedance seen between the terminals output and input
GetNoisePSD[netlist,{output,ref},source]determines the single-sided noise power spectral density caused by the source seen at the nodes output and ref
GetTotalNoisePSD[netlist,{output,ref}]determines the single-sided noise power spectral density due to all noise sources seen at the nodes output and ref
GetZVTimeConstants[netlist]uses the method of zero-value time constant analysis to determine the approximate pole locations in a netlist
